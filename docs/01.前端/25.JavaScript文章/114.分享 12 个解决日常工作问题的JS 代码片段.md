# 1、破坏赋值

在 JavaScript 中，您可以使用析构方法将数组中的值解包并将它们分配给其他变量。

```javascript
// 1. Destructive Assignment
const data = ["Paul", "too old", "Software Engineer"]
const [name, age, job_title] = data

console.log(name, age, job_title) // Paul too old Software Engineer
```
