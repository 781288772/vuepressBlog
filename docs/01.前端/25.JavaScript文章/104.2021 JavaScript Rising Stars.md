### The TOP 3

今年最流行的项目是 zx，这是 Google 的一个全新工具，可以用 JavaScript 或 TypeScript 编写简单的命令行脚本。

基本上，它允许您在代码中嵌入任何 bash 表达式（ls、cat、git...任何东西！）并使用 JavaScript 模板文字等待结果。

它包括几个流行包提供的实用程序：

- node-fetch 使用与浏览器中的 fetch 相同的 API 发出 HTTP 请求
- fs-extra 处理文件系统操作
- globby 匹配给定用户友好模式的文件名
